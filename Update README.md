# TaskFlow - Modern To-Do Application

## 📌 Project Overview

TaskFlow is a modern and responsive To-Do Application built using HTML, CSS, and JavaScript. It helps users efficiently manage their daily tasks with features such as task creation, editing, deletion, priority management, due dates, and task filtering.

## 🚀 Features

- Create New Tasks
- Edit Existing Tasks
- Delete Tasks
- Mark Tasks as Completed
- Task Priority Levels (Low, Medium, High)
- Due Date & Time Management
- Search Tasks
- Filter Tasks (All, Pending, Completed, Overdue)
- Statistics Dashboard
- Overdue Task Detection
- Clear Completed Tasks
- Local Storage Persistence
- Fully Responsive Design

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Local Storage API
